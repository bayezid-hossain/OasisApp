# Changelog

## [1.0.0]
### Added
- Voice capture with pre-warmed SpeechRecognizer (<500ms start)
- Text capture mode — type directly from the overlay
- Rule-based classification: Reminder, Idea, Contact, Note
- Local Room DB storage (offline-first)
- AlarmManager-based reminders
- 4x2 App Widget with mic shortcut
- Quick Settings Tile for lock-screen capture
- Weekly recap notification (Sunday 9 AM)
- Keyword search
- Haptic feedback on capture start and save
